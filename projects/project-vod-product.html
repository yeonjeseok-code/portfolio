온라인 VOD 콘텐츠 상품 기획

시장·사용자 데이터를 기반으로 46개 온라인 VOD 콘텐츠를 상품화한 프로젝트

디지털 드로잉·드로잉·디자인 분야에서 온라인 VOD 콘텐츠를 기획하고, 크리에이터의 전문성을 디지털 제품(Product) 관점에서 재구조화해 실제로 판매 가능한 상품으로 만들었습니다. 클래스101과 블루타이거에서 신규 VOD 상품 46개를 기획·론칭하고, 기획부터 상세페이지·운영·개선까지 전 주기를 리딩한 경험입니다.

총 46개 온라인 VOD 신규 기획·론칭 시장·리뷰 데이터 기반 주제 선정 기획·상세페이지·운영까지 End-to-End PM
프로젝트 범위
  • 클래스101(2021~2022) · 블루타이거(2023~2025)
  • 디지털 드로잉/아트/디자인 VOD 상품 46개 신규 기획
  • 시장 분석 → 기획 → 상세페이지 → 론칭 → 개선까지 전 주기
역할
  • 기획/PM 1인 주도, 크리에이터·디자인·마케팅·운영과 협업
  • 콘텐츠를 문제 해결형 디지털 Product로 재구성
  • 리뷰·데이터 기반 개선 사이클 설계 및 운영
프로젝트 개요

단순히 “강의를 하나 더 올리는 것”이 아니라, 디지털 드로잉·디자인 분야 크리에이터의 전문성을 사용자의 문제와 연결해 구매할 만한 디지털 Product로 만드는 것이 목표였습니다. 시장·리뷰·커뮤니티 데이터를 기반으로 주제를 선정하고, 상세페이지·커리큘럼·운영 모델까지 함께 설계해 실제 매출과 사용자 반응으로 검증했습니다.

프로젝트 스코프
  • 디지털 드로잉·드로잉·디자인 분야 온라인 VOD 46개 신규 기획 및 론칭
  • 클래스101: 론칭 중심 PM · 상품 기획 및 상세페이지 설계
  • 블루타이गर: 기수 기반 운영 PM · 개선 프로세스 설계·운영
  • 기획·상세페이지·운영까지 End-to-End 오너십
핵심 포인트
  • 시장을 먼저 보고 “어떤 문제를 해결하는 콘텐츠인지”를 정의한 후 기획 진행
  • 크리에이터 전문성을 스토리·작업 과정·결과물 단위로 분해해 Product화
  • Hook → 변화 → 결과물 → 학습 단계 → 보너스 구조로 상세페이지 스토리라인 설계
  • 리뷰·테스터·멘토 피드백을 기반으로 한 개선 사이클 운영
역할 & 기여도
  • 기획/PM 1인으로서 전체 온라인 VOD 상품 기획의 중심 역할 수행
  • 크리에이터/연사 1~2명과 협업하며 콘텐츠 전문성을 구조화
  • 디자인·마케팅·운영·편집 조직과의 협업 조율 및 일정 관리
  • VOD 기획부터 상세페이지·론칭·운영·개선까지 전 과정에 참여
  • 블루타이거에서는 개선 프로세스까지 포함한 운영 구조 설계까지 리딩
핵심 기여 (Key Contributions)
  1. ① 시장 기반 주제 선정 카테고리 경쟁 상황, 유사 클래스, 리뷰 패턴, 커뮤니티 인사이트를 분석해 “어떤 문제를 해결하는 클래스인지”를 먼저 정의하고 주제를 선정했습니다.
  2. ② 크리에이터 전문성 Product화 크리에이터의 스토리·작업 방식·강점을 인터뷰로 분해해, 수강생이 “왜 이 클래스를 사야 하는지”가 분명하게 보이도록 구조화했습니다.
  3. ③ 상세페이지·스토리라인 100% 기획 Hook → 변화 → 결과물 → 단계별 학습 → 보너스 요소까지 하나의 스토리라인으로 설계하고, 상세페이지 문안을 처음부터 끝까지 직접 작성했습니다.
  4. ④ 운영 PM 및 리스크 관리 론칭 일정 관리, 크리에이터·내부 팀과의 커뮤니케이션, 품질 점검을 담당하며, 기수 운영 과정에서 발생하는 리스크를 조율했습니다.
  5. ⑤ 피드백 기반 개선 구조 리뷰·테스터·멘토 피드백을 정리해 기술적 문제/커리큘럼 문제/운영 문제를 구분하고, 다음 기수(A+2)까지 반영되는 개선 사이클을 구축했습니다.
세부 업무

각 클래스는 같은 포맷으로 찍어내지 않고, 카테고리·크리에이터·사용자 세그먼트에 따라 구조를 다르게 가져가야 했습니다. 이를 위해 다음과 같은 업무를 반복적으로 수행했습니다.

1) 시장 분석
  • 카테고리별 수요·트렌드, 경쟁 클래스 구성, 가격대 분석
  • 사용자 리뷰·커뮤니티 인사이트를 통해 Pain-point·이탈 요인 파악
  • “어떤 상황의 사람이 무엇 때문에 이 클래스를 사는가?”를 중심으로 니즈 정의
2) 크리에이터 협업
  • 크리에이터의 전문성·작업 스타일·강점을 인터뷰로 정리
  • 크리에이터의 언어를 수강생이 이해하기 쉬운 메시지로 번역
  • 클래스 목적·핵심 가치·차별점을 함께 정의하고, 스토리 방향 정렬
3) 커리큘럼 기획
  • 초보자~중급자까지 따라올 수 있는 단계별 학습 구조 설계
  • 실습·프로젝트 기반으로 결과물이 남도록 흐름 구성
  • 분량·난이도·리듬을 고려해 촬영 단위와 세부 목차 설계
4) 상세페이지 기획
  • Hook → 변화 → 결과물 → 커리큘럼 → 보너스·FAQ까지 전체 스토리라인 설계
  • 이미지·영상 배치, 비교·전후 사례, 리뷰 활용 등 전환 요소 기획
  • 메인 카피·서브 카피·섹션별 설명 문안을 100% 직접 작성
5) 개선 프로세스 (블루타이거 중심)
  • 테스터·멘토·수강생 피드백을 문서화하고, 항목별 우선순위 설정
  • 기술적 문제 vs 커리큘럼 문제 vs 운영 문제로 구분해 개선 방향 정의
  • 8주 운영 → A+2 기수까지 개선 사항을 반영하는 구조 확립
성과 & 인사이트
  • 북토크 클래스, 새로운 드로잉 분야 등에서 TOP 6 및 TOP 30 내 상위 랭크 클래스 다수 런칭
  • 출시 직후 강력한 신규 유입을 확보한 3일 매출 1억+급 클래스 기획 및 론칭 경험
  • 기획 인풋 시간을 템플릿·구조화로 60% 이상 절감하고, 클래스 간 품질 편차 감소
  • 피드백 기반 개선 프로세스를 도입해, 운영·편집·기획의 부담을 역할 분리로 완화
  • 기획자 중심 PM 구조를 정립하고, 콘텐츠 상품화 템플릿을 구축해 이후 프로젝트에도 재사용
  • 크리에이터 경험을 관리하면서, 장기 협업·재런칭에 유리한 구조를 마련
개별 클래스의 매출·리뷰 데이터와 내부 지표는 비공개이므로, 포트폴리오용 페이지에서는 구조와 역할·방법론 중심으로만 재구성했습니다.
